Book table A · action stage

ある動作のどの段階かを表す

Expresses the stage an action has reached.

Book overview

動詞や名詞に別の言葉をつけると、その動詞や名詞の意味を広げることができます。

You can broaden the meaning of verbs and nouns by adding other words to them.

文法形式意味例文
～かける
～かけだ

全部～し終わっていない

Indicates that an action is not complete.

動ます ＋かける・かけだ

Book example

彼女は何か言いかけて、しまった。

Book example

母は読みかけの本をおいて、買い物に出かけた。

～きる

全部／完全に～する

Indicates completion of a task or action.

動ます ＋きる

Book example

40キロの長いを走りきった。

Book example

こんなにたくさん、一人では食べきれない。

～通す

最後まで～し続ける

Continue until completion, until the end.

動ます ＋通す

Book example

自分で決めたことは最後までやり通そう。

Book example

彼は何を聞かれても黙り通した。

～出す

（突然・急に）～し始める

Begin something suddenly.

動ます ＋出す

Book example

彼はわたしのを見ると、笑い出した。

Book example

電池を入れ替えたら、おもちゃの車が急に動き出した。

Expanded explanation

かける stops partway, きる reaches the endpoint, 通す emphasizes persistence through the endpoint, and 出す marks a sudden beginning.

Added examples

Added example

書きかけのメールを保存した。

I saved the email I had started but not finished.

Added example

長い小説を一晩で読みきった。

I finished the long novel in one night.

Added example

彼女は最後まで自分の意見を言い通した。

She maintained her opinion to the very end.

Added example

赤ちゃんが急に泣き出した。

The baby suddenly began to cry.

Book table B · characteristics and condition

特徴・状態・様子を表す①

Expressing the characteristics, condition or appearance of something.

文法形式意味例文
～やすい

簡単に～できる
簡単に～してしまう

Means that something can happen, or is done, easily; “easy to ~,” or “it can easily ~.”

動ます ＋やすい

Book example

田中先生の話はわかりやすい。

Book example

薄くて破れやすい紙だから気をつけて。

～にくい

簡単に～できない
簡単には～しない

Means that something cannot happen or be done easily; “not easy to ~,” or “it cannot easily ~.”

動ます ＋にくい

Book example

この肉は食べにくい。

Book example

丈夫で割れにくいカップはありませんか。

～づらい

～するのが難しくて困る

Means awkward or difficult to ~, causing the speaker discomfort.

動ます ＋づらい

Book example

なことなのでの人には頼みづらい。

Book example

ここはの字が見づらい位置です。

～すぎる
～すぎだ

適切な程度を超えて
～だ・～する

Means to do too much, go too far in an action or process.

動ます・イ形い・ナ形 ＋すぎる・すぎだ

Book example

この靴はわたしには小さすぎる。

Book example

おいしかったので、食べすぎてしまった。

Book example

君、それは言いすぎだよ。

Expanded explanation

にくい can describe objective resistance; づらい often adds the speaker’s physical, social, or emotional discomfort. やすい can be positive (“easy to understand”) or cautionary (“easy to tear”).

Added examples

Added example

このペンは軽くて書きやすい。

This pen is light and easy to write with.

Added example

この漢字は形が似ていて覚えにくい。

These kanji look similar and are hard to remember.

Added example

先生にはその失敗を話しづらかった。

It was uncomfortable to tell the teacher about that failure.

Added example

この道は夜でも歩きやすい。

This road is easy to walk even at night.

Added example

説明が専門的すぎてわからない。

The explanation is too technical to understand.

Added example

働きすぎないようにしてください。

Please take care not to work too much.

Book table B · tendency, image, resemblance, abundance

特徴・状態・様子を表す②

文法形式意味例文
～がちだ

～の状態になる傾向がある

Means tends toward a state or situation.

名・動ます ＋がちだ

Book example

雨のため工事は遅れがちだ。

Book example

遠慮がちに由香さんにを聞いてみた。

～らしい

～のイメージと同じだ

Means conform with a preconceived image of, be typical of.

名 ＋らしい

Book example

今日は暖かくて春らしい一日だった。

Book example

いつもの元気なリーさんらしくないですね。

～っぽい

～のような要素・性質がある

Have a certain nature or quality.

名・動ます ＋っぽい

Book example

彼女はわがままばかり言って、子どもっぽい人だ。

Book example

弟はあきっぽくて、何でも途中でやめてしまう。

～のようだ
～みたいだ

～によく似ている

Be very similar to ~.

名 ＋のようだ・みたいだ
名 ＋のような・みたいな＋名

Book example

ここから見える景色は絵のようだ。

Book example

このパンはケーキみたいに甘くて軟かい。

Book example

ガラスみたいな氷を靴で踏んで遊んだ。

～だらけだ

～（いやなもの）がたくさんある

Be in abundance, full of, littered with (usually used pejoratively).

名 ＋だらけだ

Book example

返ってきたテストは間違いだらけだった。

Book example

毎日サッカーをしていたら、傷だらけになった。

Expanded explanation

らしい says something fits its expected identity; っぽい says it merely has a noticeable quality. ようだ／みたいだ makes a comparison, while だらけ usually evaluates an unwanted abundance.

Added examples

Added example

忙しいと朝食を抜きがちだ。

When busy, I tend to skip breakfast.

Added example

今日は秋らしい涼しい日だ。

Today is a cool day typical of autumn.

Added example

このスープは少し水っぽい。

This soup is a little watery.

Added example

あの雲は動物のように見える。

That cloud looks like an animal.

Added example

駅は観光客だらけだった。

The station was packed with tourists.

Added example

彼の説明は言い訳っぽく聞こえた。

His explanation sounded rather excuse-like.

One-point lesson · source comparison

「～らしい」と「～のようだ・～みたいだ」

どちらが適当な方を選びなさい。

1　もう大人なんだから、そんな子ども（　）しゃべり方はやめろ。

2　子どもにはやはり子ども（　）服が似合いますよね。

3　6月に入ってからまだ雨（　）雨は降っていない。

4　あれ、雪（　）雨が降ってきたよ。

5　まだ5月なのに、今日は夏（　）日でしたね。

6　へえ、これが交番か。絵本に出てくるかわいい家（　）ね。

7　箱根は昔からの観光地だから、観光地（　）お土産屋が多い。

Book distinction

～らしい

実際に～だ。～のイメージと同じだ。

Means something is as it is supposed to be, as imagined.

Book example

今日は春らしい天気だった。（今は本当に春だ。）

Book example

町はお祭りらしいにぎやかさだった。（実際にお祭りだ。）

～のようだ・～みたいだ

実際に～ではないが、～によく似ている。「～みたいだ」は話し言葉。

Means not actually ~, but is very similar to ~. ～みたいだ is used in spoken language.

Book example

今日は春のような天気だった。（今は春ではない。）

Book example

町はお祭りみたいなにぎやかさだった。（実際にはお祭りではない。）

Expanded practice

Choose the form that matches the intended stage or quality.

1　まだ途中のレポートを表す形はどれですか。

3　社会的に頼むのが気まずい、という気持ちに最も近い形はどれですか。

4　忙しい人が約束を忘れる傾向を表す形はどれですか。

5　実際に冬で、冬のイメージに合う寒さです。

6　実際にはガラスではないが、ガラスによく似ています。

7　床にはいやな汚れがたくさんあります。

8　電車が急に動き（　）、立っていた人が驚いた。
