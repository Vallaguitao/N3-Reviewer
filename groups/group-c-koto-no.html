Book grammar · こと only

「こと」だけを使う場合（「の」は使わない）

Book introduction

「こと」「の」はようにと、ばならないがあります。

In some cases, こと and の may be used interchangeably, but in others they have separate, distinctive uses.

Book explanation

a）「～は…ことだ」ので、～のを…でとき

こと only is used when … explains what ～ is in a ～は…ことだ sentence.

Book example

・わたしののは、になること（の）です。（の＝になること）

・My dream for the future is to become a manga artist. (Future dream = becoming a manga artist)

Book example

・サッカーというスポーツのは、にをはいけないこと（の）だ。

・A characteristic of the sport of soccer is that you basically cannot use your hands.

Book example

・AランチとBランチのは、AがでBがであること（の）だ。

・The difference between Lunch A and Lunch B is that A is a fish dish and B is a meat dish.

Book explanation

b）「こと」を

Grammatical forms using こと.

Book formation

～ことがある　⇒「たまに～のがある。」
　／ない　＋ことがある

There are times when ~ ⇒ “Sometimes there are times when ~.” Dictionary dictionary form/no form + there are times when there is ~

Book example

①はぼくがもをしないことがある。

①My wife sometimes doesn't respond even when I talk to her.

Book example

②はが、12ごろにこともあった。

②I used to have a lot of work to do, so I would sometimes go home around 12 o'clock.

Book formation

～ことはない　⇒「～するはない・～しなくてもいい。」
　　＋ことはない

~ koto ni nai ⇒ “There is no need to ~.”

Book example

①のはですよ。そんなにすることはありませんよ。

① Interview questions are easy. There's nothing to worry about.

Book example

②があるが、ただのかぜだろう。すぐににことはない。

②I have a little fever, but it's probably just a cold. I won't go to the hospital right away.

Added examples

Added example

のはN3にすることです。

My goal is to pass N3.

Added example

でもをすることがある。

There are times when I work even on weekends.

Added example

まだがあるから、ことはない。

There is still time, so there is no need to hurry.

What matters is continuing a little every day.

Book grammar · の only

「の」だけを使う場合（「こと」は使わない）

Book explanation

a）でとらえたやなどをとき（「・・・・など」のを。）

の only is used when sounds, sights, etc. are registered by the speaker using his senses; (used with verbs such as みる, みえる, ながめる, きこえる or かんじる).

Book example

・このからでたちがいるの（こと）が。

・I can see children playing in the garden from this window.

Book example

・あの、がの（こと）を。

・That day, I felt the house shake violently.

Book explanation

b）「～のに（は）…」ので、～というについての（だ・など）を…でとき

の only is used when, in a sentence using ～のに…, the speaker wishes to express an opinion (…) about an object (～), using words such as べんりだ or やくにたつ.

Book example

・はこのでするの（こと）にどうしてもなのだ。

・A car is absolutely necessary for living in this village.

Book example

・でをするの（こと）にはファストフードがやはりだ。

・Fast food is convenient when you want to eat in a short time.

Book explanation

c）「～のは…だ」ので、したいを…でとき

の only is used when the speaker wishes to emphasize information in … in a sentence using the ～のは…だ pattern.

Book example

・にの（こと）は５である。（５にに。）

・I met her for the first time five years ago. (I met her five years ago.)

Book example

・の（こと）はがあったからだ。

・I came home late because I had to work overtime.

Book explanation

d）「の」を

Grammatical forms using の.

Book formation

～というのは…だ　⇒「～のは…だ。」
　＋というのは…だ

~ is... ⇒ "The meaning of ~ is..." Name + is...

Book example

①というのはのさがのことである。

① An equilateral triangle is a triangle whose three sides are the same length.

Book example

②「アクセスする」というのはどんなですか。

② What does "access" mean?

Book formation

～のではないか・～のではないだろうか　⇒「～と。」
（ナ　だ－な・　だ－な）＋のではないか・のではないだろうか

Isn't it...? ⇒ "I think..." Ordinary form (Na form da-na/name da-na) + I wonder if it's...

Book example

①こんなにはによくないのではないか。

①Aren't foods this salty bad for your health?

Book example

②もしかしたらヤンさんはのことをいるのではないでしょうか。

②Maybe Mr. Yang knows the truth.

Added examples

Added example

のでがいるのが。

I can hear a baby crying in the next room.

Added example

をのにこのアプリはだ。

This app is useful for memorizing kanji.

Added example

がをのはのときだ。

It was in high school that I began Japanese.

Added example

ほうがいいのではないか。

Wouldn't it be better to rest a little?

Book one-point lesson

「物」と「こと」

Book check

◎（　）の中に「物」か「こと」を書きなさい。

1　由美が作る（　）はいつもとてもおいしいね。

2　おいしい料理を作る（　）はとても楽しい。

3　子どもを育てるという（　）をわたしは大切に考えています。

4　これは何という（　）ですか。触ってみてもいいですか。

5　子どものころ祖母にいつも言われていた（　）を思い出した。

6　友だちに「持ってきてね」と言われていた（　）を家に置いてきてしまった。

7　この作文は先週自習の時間に書いた（　）です。

8　先週作文に書いた（　）は、全部本当です。

9　今日やりたいのは、ここに置いてある（　）を片付ける（　）です。

Book explanation

もの：がある、でに

Something that has shape and is physically visible to the eye.

Book example

・ちょっとください。これ、きのうです。

・Please take a look. This is what we talked about yesterday.

Book example

・かはありませんか。

・Do you have anything to eat?

Book example

・、おじができてくれたをでもにしている。

・I still cherish the things my uncle bought for me in a foreign country a long time ago.

Book explanation

こと：がなく、で

Something lacking shape, and invisible to the eye.

Book example

・きのうわたしがことはですよ。

・Everything I said yesterday is true.

Book example

・わたしたちだけでおいしいをことは、にはいようね。

・Let's keep quiet from Dad about eating delicious food alone.

Book example

・、おじがからおをきてくれたことをよくいる。

・I remember well when my uncle bought me a souvenir from a foreign country a long time ago.
