Book grammar · こと only

「こと」だけを使う場合（「の」は使わない）

Book introduction

「こと」「の」は両方同じように使える場合と、使い分けなければならない場合があります。

In some cases, こと and の may be used interchangeably, but in others they have separate, distinctive uses.

Book explanation

a）「～は…ことだ」の文で、～の内容を…で示すとき

こと only is used when … explains what ～ is in a ～は…ことだ sentence.

Book example

・わたしのの夢は、になること（の）です。（将来の夢＝漫画家になること）

Book example

・サッカーというスポーツのは、基本的に手を使ってはいけないこと（の）だ。

Book example

・AランチとBランチの違いは、Aが魚料理でBが肉料理であること（の）だ。

Book explanation

b）「こと」を使う文法形式

Grammatical forms using こと.

Book formation

～ことがある　⇒「たまに～の場合がある。」
動　辞書形／ない形　＋ことがある

Book example

①妻はぼくが話しかけてもをしないことがある。

Book example

②以前は仕事が多くて、12時ごろ家に帰ることもあった。

Book formation

～ことはない　⇒「～する必要はない・～しなくてもいい。」
動　辞書形　＋ことはない

Book example

①の質問は簡単ですよ。そんなに心配することはありませんよ。

Book example

②少し熱があるが、ただのかぜだろう。すぐに病院に行くことはない。

Book cross-references

～ということだ・～とのことだ　→第１部 ７課－①
～ないことはない　→第１部 ８課－④
～ことは～が、…　→第１部 ８課－⑤
～こと　→第１部 10課－②
～ことにする・～ことにしている　→第１部 11課－①、第１部G
～ことになる・～ことになっている　→第１部G

Added examples

Added example

私の目標はN3に合格することです。

My goal is to pass N3.

Added example

週末でも仕事をすることがある。

There are times when I work even on weekends.

Added example

まだ時間があるから、急ぐことはない。

There is still time, so there is no need to hurry.

Added example

大切なのは毎日少しずつ続けることだ。

What matters is continuing a little every day.

Book grammar · の only

「の」だけを使う場合（「こと」は使わない）

Book explanation

a）でとらえた音や光景などを言うとき（「見る・見える・眺める・聞こえる・感じるなど」の動詞を使う。）

の only is used when sounds, sights, etc. are registered by the speaker using his senses; (used with verbs such as みる, みえる, ながめる, きこえる or かんじる).

Book example

・この窓から庭で子どもたちが遊んでいるの（こと）が見える。

Book example

・あの日、家が大きく揺れるの（こと）を感じた。

Book explanation

b）「～のに（は）…」の文で、～というについての評価（便利だ・役に立つなど）を…で言うとき

の only is used when, in a sentence using ～のに…, the speaker wishes to express an opinion (…) about an object (～), using words such as べんりだ or やくにたつ.

Book example

・車はこの村で生活するの（こと）にどうしても必要なのだ。

Book example

・短時間で食事をするの（こと）にはファストフードがやはり便利だ。

Book explanation

c）「～のは…だ」の文で、強調したい情報を…で示すとき

の only is used when the speaker wishes to emphasize information in … in a sentence using the ～のは…だ pattern.

Book example

・彼女に初めて会ったの（こと）は５年前である。（５年前に彼女に会った。）

Book example

・遅く帰ったの（こと）はがあったからだ。

Book explanation

d）「の」を使う文法形式

Grammatical forms using の.

Book formation

～というのは…だ　⇒「～の意味は…だ。」
名　＋というのは…だ

Book example

①というのは三辺の長さが同じ三角形のことである。

Book example

②「アクセスする」というのはどんな意味ですか。

Book formation

～のではないか・～のではないだろうか　⇒「～と思う。」
普通形（ナ形　だ－な・名　だ－な）＋のではないか・のではないだろうか

Book example

①こんなに食品は体によくないのではないか。

Book example

②もしかしたらヤンさんは本当のことを知っているのではないでしょうか。

Added examples

Added example

隣の部屋で赤ちゃんが泣いているのが聞こえる。

I can hear a baby crying in the next room.

Added example

漢字を覚えるのにこのアプリは便利だ。

This app is useful for memorizing kanji.

Added example

私が日本語を始めたのは高校生のときだ。

It was in high school that I began Japanese.

Added example

少し休んだほうがいいのではないか。

Wouldn't it be better to rest a little?

Book one-point lesson

「物」と「こと」

Book check

◎（　）の中に「物」か「こと」を書きなさい。

1　由美が作る（　）はいつもとてもおいしいね。

2　おいしい料理を作る（　）はとても楽しい。

3　子どもを育てるという（　）をわたしは大切に考えています。

4　これは何という（　）ですか。触ってみてもいいですか。

5　子どものころ祖母にいつも言われていた（　）を思い出した。

6　友だちに「持ってきてね」と言われていた（　）を家に置いてきてしまった。

7　この作文は先週自習の時間に書いた（　）です。

8　先週作文に書いた（　）は、全部本当です。

9　今日やりたいのは、ここに置いてある（　）を片付ける（　）です。

Book explanation

物もの：形がある、目で実際に見える

Something that has shape and is physically visible to the eye.

Book example

・ちょっと見てください。これ、きのう話した物です。

Book example

・何か食べる物はありませんか。

Book example

・昔、おじが外国で買ってきてくれた物を今でも大切にしている。

Book explanation

こと：形がなく、目で見えない

Something lacking shape, and invisible to the eye.

Book example

・きのうわたしが話したことは全部本当ですよ。

Book example

・わたしたちだけでおいしい物を食べることは、お父さんには黙っていようね。

Book example

・昔、おじが外国からお土産を買ってきてくれたことをよく覚えている。
