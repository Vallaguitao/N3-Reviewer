Book grammar · natural conclusion

～わけだ・～というわけだ

Book formation

普通形（ナ形 だ－な／－である・名 だ－の／－な／－である）＋わけだ
普通形（ナ形（だ）・名（だ））＋というわけだ

Book explanation

「事情から考えると、当然～という結論になる。（①②③）」「事情から考えると～という事実が納得できる。（④⑤）」

Used when an outcome arises naturally from a given set of circumstances, as in ①, ② and ③, or to indicate acceptance of a fact in light of a certain set of circumstances, as in ④ and ⑤.

Book example

①ここからまで１時間半か。じゃ、今から出れば９時には着くわけだ。

Book example

②ほかに空いている日がなかったから、その日に会うことにしたわけです。

Book example

③金曜日は授業が休み、月曜日は祝日だ。つまり、４というわけだ。

Book example

④に雪が降ったんですね。それで、きのうの夜あんなに寒かったわけですね。

Book example

⑤彼女のお父さんはですか。それで、彼女も絵が上手だというわけなんですね。

Expanded explanation

Use this when the earlier facts make the conclusion natural. It can also show “now I understand why” after receiving an explanation.

Added examples

Added example

毎日三時間練習したのか。上手になるわけだ。

You practiced three hours every day. No wonder you improved.

Added example

今日は祝日だから、銀行は休みというわけです。

Today is a holiday, so naturally the bank is closed.

Added example

電車が止まっていたんですか。それで遅れたわけですね。

The train had stopped? So that explains why you were late.

Added example

一袋に十個入っているから、三袋で三十個になるわけだ。

There are ten in each bag, so three bags make thirty.

Book grammar · restrained action

～わけにはいかない

Book formation

動　辞書形 ＋わけにはいかない

Book explanation

「心理的事情があるので、～することはできない。」能力などが原因でできないのではなくて、～したいが、社会的常識に反する・心理的抵抗感があるなどの事情があってできないという意味で使う。主語はふつう一人称。

Means “because a certain state of mind exists, ～ cannot be done.” Not used to mean that something cannot be done for reasons of inability, but indicates that although the speaker wishes to act (～), he is held back by fear of offending social norms, or by personal reservations. The subject is usually the first person.

Book example

①がお金を貸してほしいと言っている。親友のを断るわけにはいかない。

Book example

②今日は車で来たんです。お酒を飲むわけにはいきません。

Book example

③かぜをひいてしまったが、大事ながあるから、会社を休むわけにはいかない。

Expanded explanation

The action is physically possible, but responsibility, relationships, safety, or social expectations make it unacceptable.

Added examples

Added example

明日は試験だから、今夜遊ぶわけにはいかない。

I have an exam tomorrow, so I cannot allow myself to go out tonight.

Added example

お客様を待たせるわけにはいきません。

We cannot keep the customer waiting.

Added example

借りた本なので、なくすわけにはいかない。

It is a borrowed book, so I must not lose it.

Added example

約束した以上、途中でやめるわけにはいかない。

Since I promised, I cannot quit halfway.

Book grammar · unavoidable action

～ないわけにはいかない

Book formation

動　ない形 ＋わけにはいかない

Book explanation

「心理的事情があるので、～しなければならない。」規則などで決まっていて～しなければならないのではなくて、あまり～したくないが、社会的常識・心理的義務感があってそうする必要があるという意味で使う。主語はふつう一人称。

To have to ～ because a certain state of mind exists. Not used when the speaker must ～ because of rules, but when the speaker must (reluctantly) do so to conform with public norms or out of a sense of personal obligation. The subject is usually the first person.

Book example

①この町ではがないとやはり困る。買わないわけにはいかない。

Book example

②本当に暑いですけど、何も着ないわけにはいきませんよね。

Book example

③このCD、ずっと持っていたいけど、のだから返さないわけにはいかない。

Book cross-references

～わけがない　→第１部８課－①
～わけではない・～というわけではない　→第１部８課－③

Added examples

Added example

招待されたので、行かないわけにはいかない。

I was invited, so I cannot avoid going.

Added example

間違えたのは私だから、謝らないわけにはいかない。

I made the mistake, so I have to apologize.

Added example

みんなが手伝っているから、私も働かないわけにはいかない。

Everyone is helping, so I have to work too.

Added example

期限が今日なので、レポートを出さないわけにはいかない。

The deadline is today, so I have no choice but to submit the report.

Book one-point lesson · page 77

「～はずだ」と「～わけだ」

◎（　）の中に「はず」か「わけ」を書きなさい。

1　のやり方を変えてみたんです。今度はきっと成功する（　）です。していてください。

2　実験のやり方を変えてみたんです。だから、こんなにいいが出た（　）です。

3　A「はさみはどこ？」
B「いつもの引き出しの中にある（　）だよ。よく探してみて。」

4　彼は「必ず行くよ。」と言っていたのだから、来る（　）ですよ。もう少し待ちましょう。

5　山川さんもこの大会の準備係ですか。それで、こんなに早く来た（　）ですね。

6　A「今日、川口さんも会に出席するでしょうか。」
B「え？　川口さんは今、日本にはいない（　）ですよ。おとといアメリカに行ったんです。」

Book explanation · ～はずだ

話者のな推量を言う。のある推量。

Expresses the speaker’s subjective evaluation, with a high degree of confidence.

Book example

・いい薬を使ったから、きっとすぐ治るはずですよ。

Book example

・この商品は人気があるから、十日ぐらいで売り切れるはずだ。

Book explanation · ～わけだ

推量ではなくて、当然であると納得したことを言う。

Expresses the speaker’s compliance with or understanding of something as a matter of course, without any process of consideration.

Book example

・なるほど、この薬を使えば早く治るわけですね。じゃあ使ってみましょう。

Book example

・この薬は１日１袋飲むのですから、十日間で10袋になるわけですね。
