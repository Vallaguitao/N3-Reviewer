Book grammar · discovery with slight surprise

～と…た・～たら…た　A

Book overview

「たら・ば・と・なら」には、「もし…」という仮定条件の意味以外にもいろいろな使い方があります。

たら, ば, と and なら have many additional functions to their hypothetical meaning of もし.

Book formation

動　辞書形 ＋と
動　たら

Book explanation

「～の動作をすることで、…という事実に気がついた。」…は前から続いている状態を表す文で、少し意外感がある内容。「…ていた」の形がよく使われる。～と…の主語は違う。

After ～ (verb), suddenly notice … (with slight surprise). … is a preexisting state. The …ていた form is often used. The subject of the ～ clause differs from the subject of the … clause.

Book example

A①プレゼントを開けると、が入っていた。

Book example

②玄関を出ると、そこにが数人いた。

Book example

③窓の外を見たら、真っ白なだった。

Expanded explanation

The first action opens a scene in which the speaker discovers an already-existing fact. The second clause therefore often uses ていた or another state expression.

Added examples

Added example

ドアを開けると、猫が寝ていた。

When I opened the door, I found the cat sleeping.

Added example

駅に着いたら、電車はもう出ていた。

When I arrived at the station, I found the train had already left.

Added example

箱を開けると、中は空だった。

When I opened the box, I discovered it was empty.

Added example

外へ出たら、道がぬれていた。

When I went outside, I noticed the road was wet.

Book grammar · chance event

～と…た・～たら…た　B

Book formation

動　ている ＋と
動　ていたら

Book explanation

「～の動作をしているときに、…が起こった。」～は「～ていると・～ていたら」の形が多い。…は動詞の文で、話者の意志を含まない、意外感がある内容。

While doing ～ (verb), … happened by chance too. The～clause often uses the ～ていると or ～ていたら form. … is a verb phrase, and does not express the will of the speaker. Usually the event (…) is somewhat surprising.

Book example

B①夜12時ごろテレビを見ていると、アメリカの友だちから電話がかかってきた。

Book example

②コーヒーショップでマキさんのうわさをしていると、偶然マキさんが入ってきた。

Book example

③庭の掃除をしていたら、が降り出した。

Added examples

Added example

歩いていると、昔の友だちに会った。

While I was walking, I happened to meet an old friend.

Added example

料理をしていたら、電話が鳴った。

While I was cooking, the phone unexpectedly rang.

Added example

公園で本を読んでいると、雨が降ってきた。

While I was reading in the park, it suddenly began to rain.

Added example

電車を待っていたら、先生が来た。

While I was waiting for the train, my teacher happened to arrive.

Book grammar · action as trigger

～と…た・～たら…た　C

Book formation

動　辞書形 ＋と
動　たら

Book explanation

「～の動作がになって、…が起こった。」…は動詞の文で、話者の意志を含まない文。

As a result of ～ (verb), … happened. … is a verb phrase, and does not express the will of the speaker.

Book example

C①久しぶりに高校の先生に手紙を出すと、すぐにが来た。

Book example

②箱を開けると、おもちゃが飛び出した。

Book example

③料理にちょっとお酒を入れてみたら、いい味になった。

Added examples

Added example

ボタンを押すと、音楽が流れた。

When I pressed the button, music played as a result.

Added example

窓を開けたら、風が入ってきた。

When I opened the window, air came in.

Added example

薬を飲むと、熱が下がった。

Taking the medicine caused the fever to go down.

Added example

名前を呼んだら、犬が走ってきた。

When I called its name, the dog ran over.

Book grammar · immediate same-subject sequence

～と…た

Book formation

動　辞書形 ＋と

Book explanation

「～の動作の後、続けてすぐ…した。」～と…の主語は同じ。

After ～ (verb), (sentence subject) immediately … (verb). The subject remains the same.

Book example

①その男はわたしの顔を見ると、すぐに逃げていってしまった。

Book example

②先生は教室に入ってくると、すぐを配り始めた。

Book example

③ゆき子はから降りると、どこかに電話をかけた。

Expanded explanation

Unlike the discovery use, the same person performs both actions. The second action follows immediately and is narrated as a completed event.

Added examples

Added example

田中さんは席に着くと、すぐ仕事を始めた。

Mr. Tanaka sat down and immediately began work.

Added example

彼は手紙を読むと、部屋を出た。

He read the letter and immediately left the room.

Added example

子どもは家に帰ると、かばんを置いた。

The child arrived home and immediately put down the bag.

Added example

社長は説明を聞くと、すぐ質問した。

The president heard the explanation and immediately asked a question.

Book grammar · paired qualities and facts

～も…ば～も…・～も…なら～も…

Book formation

名＋も ＋［動　ば形／イ形　～ければ／ナ形　なら／名　なら］＋名＋も

Book explanation

「～も…し、～も…・～も～も両方…。」「も」を２回使う。

These constructions mean “both ～ and ….” も appears twice.

Book example

①には楽しい時もあればつらい時もある。

Book example

②妻はお酒もよく飲めば甘い物もたくさん食べる。

Book example

③そんな人、会ったこともなければ名前を聞いたこともありません。

Book example

④松本さんはが多い。スポーツも好きならピアノも弾くらしい。

Book example

⑤前回も雨なら今回も雨か……。ぼくたちの旅行はいつも運が悪いね。

Added examples

Added example

この町は海もあれば山もある。

This town has both the sea and mountains.

Added example

彼は英語も話せれば中国語も話せる。

He can speak both English and Chinese.

Added example

この店は値段も安ければ味もいい。

This shop has both low prices and good food.

Added example

兄は料理も上手なら掃除も得意だ。

My older brother is both good at cooking and good at cleaning.

Book one-point lesson · page 91

仮定を表す「～たら…・～ば…・～と…」の注意

◎どちらか適当な方を選びなさい。（両方いい場合もあります。）

1　車を（a 貸してもらえると　b 貸してもらえれば）、ドライブに行きたい。

2　早く仕事が終わったら、（a 飲みに行けるね　b 飲みに行きませんか）。

3　早く予約すれば、（a いい席を取る　b いい席が取れる）。

4　部屋が暑ければ、（a 窓を開けてください　b 窓を開けますよ）。

5　時間が（a なければ　b なかったら）、買い物はあきらめるつもりだ。

6　マフラーを（a 作れば　b 作ったら）、彼にあげようと思う。

7　店の（a 電話番号を調べれば　b 電話番号がわかれば）、電話してみます。

Book explanation · movement verbs

※～が動きを表す動詞の場合

When the verb of the たら phrase expresses movement.

「～たら…」の…はどんな文でもいいが、「～ば…・～と…」の…には話者の意向を表す文や働きかけの文は来ない。

The … of ～たら… clauses be any kind of statement, but the … in ～ば… or ～と… clauses cannot express the speaker's intentions or inducements of others.

Book example · ×

×　大雪が降ると家にいよう。

Book example · ×

×　大雪が降れば家にいなさい。

Book example · ○

○　大雪が降ると／降れば出かけられない。

Book explanation · adjectives and state verbs

※～が形容詞や状態を表す動詞（ある・いる・できるなど）の場合

When the predicate of the たら phrase is an adjective or a verb of state (ある, いる, できる, etc).

「～たら…・～ば…」の…はどんな文でもいいが、「～と…」の…には話者の意向を表す文や働きかけの文は来ない。

The … of ～たら… and ～ば… clauses can be any kind of statement, but the … of ～と… clauses cannot express the speaker's intentions or inducement of others.

Book example · ×

×　お金があると車を買うつもりだ。

Book example · ○

○　お金があれば車を買いたい。

Book example · ○

○　お金があると／あれば車が買える。
