Book grammar · limitation

～ばかり…

Book formation

（＋）・　／ている ＋ばかり

Noun (+particle)/verbal dictionary form/iteru +bakari

Book explanation

「いつも～だけで、ほかのもの・ことはない。」よくないといることをが。

Always do ～ only, do nothing but ～, there is nothing but ～. Often used to express disapproval.

Book example

①はあきずにカップラーメンばかりいる。

①My younger brother eats nothing but cup ramen every day.

Book example

②ではのとばかり、いろいろなのとしたほうがいい。

②In the dormitory, it's better to have conversations with people from various countries instead of just talking to people from the same country.

Book example

③ありがとうございます。いつもいただくばかりで、もできなくてすみません。

③Thank you. I'm sorry for always giving you so much and not being able to give anything in return.

Book example

④はただいるばかりで、があったのかわからなかった。

④The child was just crying and had no idea what had happened.

Book example

⑤こののはどうしているのでしょう。のをばかりです。

⑤What is the girl in this photo doing now? I just wish her happiness.

Expanded explanation

The form narrows attention to one repeated item or action. Context often gives it a critical tone, though an action such as 祈るばかり can simply mean “all one can do is pray.”

Added examples

Added example

のはばかりいる。

On my days off, I watch nothing but videos.

Added example

をばかりで、も。

He only complains and does not help at all.

Added example

ばかりほうがいい。

You should not eat only sweets.

Added example

あとはをばかりです。

All we can do now is wait for the result.

Book grammar · repeated behavior

～てばかりいる

Book formation

　て ＋ばかりいる

There is only a movement + shape.

Book explanation

「ほかのことはしないで、よく～する。」ので。

Repeatedly or continuously ～, without doing other things. Usually used in a critical way.

Book example

①はばかりいる。

① My grandfather has been getting angry a lot lately.

Book example

②のころはばかりいた。しなかったことをはにいる。

②When I was 20, all I did was play. I now regret not studying.

Book example

③ただばかりいないで、はくださいよ。

③Don't just watch, please help me a little.

Added examples

Added example

はゲームをしてばかりいる。

My brother does nothing but play games.

Added example

してばかりいないで、まずしよう。

Do not just worry; let us act first.

Added example

、がばかりいる。

Recently it has been raining constantly.

Added example

はのをしてばかりいて、のを。

He keeps talking about himself and does not listen to others.

Book grammar · addition

～ばかりでなく…

Book formation

（＋）・（ナ だ－な／－である・ だ－である）＋ばかりでなく

Not only name (+ particle), ordinary form (na form da-na/- desu, name da- de deru) +

Book explanation

「～だけでなくて、そのほかにも～。」

Not just ～, in addition there is ～.

Book example

①ばかりでなく、のがエネルギーにをいる。

① Not only Japanese people but people all over the world are interested in energy issues.

Book example

②このは、ばかりでなく、さまざまなことが。

②This program is not only interesting, but you can learn various things.

Book example

③はをしてくれたばかりでなく、こののもごちそうしてくれた。

③He not only showed me around the town, but also treated me to local cuisine.

Added examples

Added example

はばかりでなく、も。

She speaks not only English but also Chinese.

Added example

このはばかりでなく、サービスもいい。

This store is not only inexpensive but also has good service.

Added example

はをばかりでなく、もしてくれた。

Mr. Tanaka not only prepared the materials but also gave the presentation.

Added example

はばかりでなく、にもいい。

Exercise is good not only for the body but also for the mind.

Book grammar · change and readiness

～ばかりだ

Book formation A

　 ＋ばかりだ

Formation: verb dictionary form + ばかりだ.

Book explanation A

「～というにがいく。」～はを（なる・など）。よくないのが。

Used when a change (よわくなる or へる, etc.) is ongoing. Usually used to express a worsening trend.

Book example

A①けんかしてから、とはがなるばかりだ。

A①After we had a fight once, my relationship with my girlfriend only got worse.

Book example

②はいつもいなければいくばかりだ。

② If you don't use a foreign language all the time, you'll likely forget it.

Book example

③、はがなるばかりでです。

③Recently, my grandmother's body has been getting weaker and I am worried.

Book formation B

　 ＋ばかりだ

Formation: verb dictionary form + ばかりだ.

Book explanation B

「が、はただ～するだけのだ。」

All is ready, and it only remains to ～.

Book example

B①のはできました。もうするばかりです。

B① I am ready for the trip. I'm about to leave.

Book example

②のが、もうばかりになっている。

②I have finished preparing the meal and am now ready to eat.

Book example

③パーティーのが、はにばかりというときになって、ミスが。

③After the party invitations were completed and all that was left to do was send them to the guests, I discovered a mistake.

Added examples

Added example

はばかりだ。

Prices just keep rising.

Added example

のはばかりで、がだ。

The river water keeps decreasing, so rain is needed.

Added example

は。あとはするばかりだ。

Everything is loaded; all that remains is to depart.

Added example

はして、するばかりになっている。

The presentation materials are complete and only need to be printed.

Book grammar · recent completion

～たばかりだ

Book formation

　た ＋ばかりだ

Formation: verb た-form + ばかりだ.

Book explanation

「～したすぐだ。」

Immediately after ～, or has just ～.

Book example

①さっきをばかりなのに、もうおなかがすいてしまった。

①Even though I just ate rice, I'm already hungry.

Book example

②したばかりなので、まだにいない。

②I just got married last month, so I'm still not used to my new life.

Book example

③ばかりのおもちゃがもうこわれてしまった。

③The toy I just bought is already broken.

Expanded explanation

The time does not have to be objectively short. The speaker presents it as recent relative to the situation.

Added examples

Added example

にばかりで、まだにいない。

I have only just come to Japan and am not used to the trains yet.

Added example

このはばかりです。

I bought these shoes only yesterday.

Added example

をばかりなのに、もう。

I have only just started the job, but I am already busy.

Added example

ばかりのをで。

I used a word I had just learned in conversation.

Book one-point lesson · page 83

「～たばかり」と「～たところ」

◎（　）の中に「ばかり」か「ところ」を書きなさい。

1　うちには生まれた（　）の子犬が３匹います。

2　会議が始まった（　）に高橋さんが入ってきた。

3　もしもし、今新幹線に乗った（　）です。そちらに８時に着くと思います。

4　このパソコンはまだ買った（　）なのに、調子が悪い。

5　彼は１か月前に日本に来た（　）だそうです。でも、日本語が上手ですね。

6　好きな曲を聞き終わった（　）で、ちょうど昼休みが終わった。

7　この４月に入社した（　）なので、まだ会社の人間関係がよくわからない。

Book explanation · ～たばかり

のだといるときに。にでなくても。に「の」をつけて、「～たばかりの」のでこともできるが、「に・へ・を」をつけてことはできない。

Used just after something has happened (in the speaker’s view). Can be used when some time has passed too. It is also possible to affix の to make the ～たばかりの form, but particles, etc. such as に, へ or を cannot be affixed.

Book example

・でき上がったばかりのケーキをみんなで食べた。

Book explanation · ～たところ

のだとときに。にでなければ。に「に・へ・を」をつけてこともできるが、「の」をつけて「～たところの」のでことはできない。

Used to mean just after something, in the speaker’s view. Can only be used when it is directly afterwards. Can be used with addition of particles に, へ or を, but ～たところの (that is, with an affixed の) is impossible.

Book example

・もしもし、今、駅に着いたところです。

Book example

・ケーキができ上がったところへ子どもたちが帰ってきた。
