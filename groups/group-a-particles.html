1. こそ

Book introduction

にいろいろなをにはのようなものがあります。

Particles used within sentences can perform a range of functions, as shown below.

ParticleMeaningExamples
こそ

ほかとはっきりしてする

Emphatic particle corresponding to “precisely” when used to underline and differentiate a point.
Book example

I want to win this time.

Book example

このこそいたものだ。

This material is what I was looking for for a long time.

Book example

It is precisely because they are parents that they scold their children harshly.

Added examples

2. でも

ParticleMeaningExamples
でも

なを、ほかはだと

Used to imply or show by extreme example that something is obvious.
Book example

そんなことはでもいる。

Even children know this.

Book example

ミスでもはいけない。

Don't overlook even small mistakes.

Book example

はとでもすぐなる。

My sister quickly becomes friends with people even when she meets them for the first time.

でも

・・などのでを

Used in proposals, expressions of intent or requests, meaning “even if it is only …”
Book example

でもか。

Shall we have some tea?

Book example

でもかな。

Maybe I should watch a movie.

Book example

はのにでもおいてください。

Please leave your luggage on the desk.

Added examples

Added example

そのはでも。

Even an elementary-school student can read that kanji.

Added example

でもは。

I eat breakfast even on busy days.

Added example

はでもしませんか。

Would you like to take a walk in a park or somewhere this weekend?

Added example

でもください。

If you are tired, please take a short break or something.

3. も

ParticleMeaningExamples
も

「～ない」とする

Emphatic negative, meaning “not even ~” or “not ~ at all.”
Book example

１もに。

I went to school without missing a day.

Book example

はだ。１もいない。

My wallet is empty. I don't have a single yen left.

Book example

この、だれにもくださいよ。

Please don't show this photo to anyone.

も

なを、ほかもだと

Used with extreme examples “even.”
Book example

がこともできない。

My legs hurt and I can't even stand.

Book example

このはもうも。

This child can already write difficult kanji.

Book example

こんなににもがある。

There is a store even on top of such a high mountain.

Added examples

Added example

きのうはも。

I could not sleep even one hour yesterday.

Added example

Added example

Book table · Unexpected extreme and minimum condition

4. さえ

ParticleMeaningExamples
さえ

なを、のをする

Used with extreme examples that underline unexpected degree; “even.”
Book example

にはさえいない。

There aren't even eggs in the refrigerator.

Book example

なはにさえことがある。

He loves traveling and has even been to the North Pole.

Book example

でさえがに。

There were questions on the exam that even academics couldn't solve.

さえ

を　→１ ６-③

Indicates a minimum condition necessary (for something to happen); “if only.”
Book example

さえよければ、それでいいのですか。

Is it okay as long as you are okay with yourself?

Book example

さえば、ができる。

As long as it doesn't rain, we can have fireworks.

Book example

にをさえすれば、にならないよ。

As long as you ask someone for directions, you won't get lost.

Added examples

Added example

はのさえいた。

He had forgotten even his own birthday.

Added example

さえ。

I was so busy that I could not even eat lunch.

Added example

パスポートさえあれば、できます。

If only you have a passport, you can complete the procedure.

Added example

しさえすれば、になります。

If only you practice ten minutes every day, you will improve.

6. ぐらい・くらい

ParticleMeaningExamples
ぐらい
くらい

・を

Denotes a minimum level or extent.
Book example

なあいさつぐらいならで。

I can say simple greetings in Japanese.

Book example

はくらいおをもいいね。

I don't mind having a few drinks today.

Book example

ぐらいくださいよ。

Please give me a break on Sunday.

ぐらい
くらい

のを　→１ ３-①

Used to express similarity of extent or scale, by citing a similar example.
Book example

ぐらいのさのパンを。

I made a loaf of bread about the size of an egg.

Book example

うちのぐらいのがいた。

A girl about my daughter's age was crying.

Book example

このはわたしにもくらいのだ。

This car is cheap enough for me to buy.

Added example

Added example

このはでもくらいだ。

ParticleMeaningExamples

なんか

Used to suggest something or raise alternatives; means “or something like that.”
Book example

How about this clothes? It suits you.

Let's go somewhere for the holidays. How about Hawaii?

Book table · Limitation and range

8. だけ

ParticleMeaningExamples
だけ

する

Expresses limitation; “only.”
Book example

わたしはがだが、へびだけはいやだ。

I like animals, but I don't like snakes.

Book example

にだけはのことをと。

I think I'll just tell my mother the truth.

Book example

はだけでも。

He just listened and didn't say anything.

だけ

のを

“Only,” in the sense of limited range.
Book example

なだけもいいよ。

You can eat as much as you like.

Book example

はだけしまった。

He said whatever he wanted to say and went home.

Book example

やれるだけのことはもうみんなやった。

We've all done all we can.

Book note

＊これらのは、のようなをするのにもつく。　→１ B

These particles can also be appended to words that function like particles (Part 1-B).

Book example

・わたしはののについてさえよく。

・I don't even know much about the history of my country.

Book example

・はまでを。

・Kumi uses honorific language even to her closest friends.

Added examples

Added example

なだけください。

Only the people who need to should enter.

Added example

このことはにだけ。

I told only my family about this.

Added example

なだけしてください。

Please ask as many questions as you like.

Added example

できるだけのは。

I finished all the work I could do today.

Book one-point lesson

「も」と「しか」、「ぐらい・くらい」と「まで」

Book check

◎（　）の中に「も」か「しか」を書きなさい。

1　ここに住所を書くんですか。わたしはまだひらがな（　）書けません。ひらがなでいいですか。

2　ここに住所を書くんですか。わたしはまだひらがな（　）書けません。ローマ字でいいですか。

3　A「胃の検査をするので、朝から水（　）飲んでいないんです。」
B「あ、水を飲んだんですか。それじゃ、検査ができないと思いますよ。」

4　A「答えがわかった人はクラスで一人（　）いなかったんです。」
B「え！ だれもわからなかったんですか。」

Book check

◎（　）の中に「ぐらい」か「まで」を書きなさい。

5　母は30年も前の細かいこと（　）よく覚えている。

6　きのうのこと（　）忘れないで覚えていてくださいよ。

7　自分のこと（　）自分で考えなさい。

8　10年も前にもらった年賀状（　）大事にしまってある。

Book explanation

「も」と「しか」：「～も」は～ををする。「～しか」は～をする。

～も means “not a single ~,” including the ~ phrase, while ～しか means “everything else except the ~ phrase.”

Book example

・もひまはない。

I do not have a single day off.

Book example

・しかひまはない。

I only get a single day off.

Book explanation

「ぐらい・くらい」と「まで」：「ぐらい」は、「まで」はを。

ぐらい denotes minimal ability (“can at least do something”), while まで denotes the opposite (“can even do something”).

Book example

・ぐらいよ。

I can (at least) cook fried eggs.

Book example

・はまでで。

Mother can even make tofu.
