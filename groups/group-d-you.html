Book grammar · resemblance

～（かの）ようだ・～のようだ・～（かの）ように…・～のように…

Book introduction

「よう」を使った文法形式は「ことを表すもの」と「期待すること・要求することを表すもの」の二つの意味に分けられます。

There are two main uses for よう: to express similarity, and to express expectation of or a demand for something.

Book meaning

⇒よく似ているものに例えて言う

Used to illustrate how something resembles something else, or to illustrate by metaphor.

Book formation

名の ＋ようだ・ように
名だ－である ＋かのような・かのように
動　普通形 ＋（かの）ようだ・（かの）ように

＊③は名詞につく形。

Book example

①今日は、まるで春が来たかのようだ。

Book example

②朝から晩までロボットのように働いた。

Book example

③バケツをような雨だった。

Expanded explanation

ようだ ends a statement, ような modifies a noun, and ように modifies a verb or describes manner. Adding かの makes an imagined comparison especially vivid.

Added examples

Added example

彼は魚のように速く泳ぐ。

He swims quickly like a fish.

Added example

町全体が眠っているかのように静かだった。

It was quiet as if the entire town were asleep.

Added example

雲のような白い犬が走ってきた。

A white dog like a cloud came running.

Book grammar · comparison and examples

～ように…

Book meaning A

⇒大体同じであることを表す

Indicates that something is roughly the same.

Book formation

名の・普通形（ナ形 だ－な／－である・名 だ－である）＋ように

＊③は名詞につく形。

Book example

①人間のように、植物にもが必要だ。

Book example

②母親が明るい人だったように、その娘たちも性格が明るい。

Book example

③わたしたちはあなたがしているような関係ではありませんよ。

Book meaning B

⇒例を示す

Cite as an example.

Book formation

名の ＋ように

＊②③は名詞につく形。

Book example

①日本語のように、使う文字が３種類もある言語は。

Book example

②わたしはにんじんやピーマンのような色の野菜が好きだ。

Book example

③林さんは優しい。林さんのような人とつき合いたい。

Expanded explanation

The first use compares two cases as broadly alike. The second presents one noun as a representative member of a larger category.

Added examples

Added example

父が毎朝走るように、私も運動を続けている。

Just as my father runs every morning, I also keep exercising.

Added example

京都のように、古い建物が多い町を訪れたい。

I want to visit a city with many old buildings, such as Kyoto.

Added example

田中さんのような親切な先生に習いたい。

I want to learn from a kind teacher like Mr. Tanaka.

Book grammar · known information and hoped result

～ように…

Book meaning A

⇒…の内容がすでに知られていると前置きする

Used at the front of a sentence, indicates that something is already understood or known.

Book formation

動　辞書形／た形／ている ＋ように

＊③は慣用的な言い方なので接続は例外的。

Book example

①前にも話したように、来週はわたしは日本にいません。

Book example

②今朝のに書いてあったように、今年は米のできがいいらしい。

Book example

③ご存じのように、日本はが多い国です。

Book meaning B

⇒期待することを表す。

Expresses a hope or wish.

Book formation

動　辞書形／ない形 ＋ように

＊話者の意志を表さない動詞を使う。

Book example

①よく眠れるようにワインを少し飲んだ。

Book example

②池田さんは難しいをだれにでもわかるように説明する。

Book example

③が目を覚まさないようにテレビの音を小さくした。

Expanded explanation

Known-information ように points backward to shared knowledge. Result-oriented ように points forward to a state the speaker wants to make possible or prevent.

Added examples

Added example

先生が説明したように、この形は書き言葉でよく使われる。

As the teacher explained, this form is often used in writing.

Added example

画面に表示されているように、受付は終了しました。

As shown on the screen, registration has ended.

Added example

忘れないように、予定をカレンダーに入れた。

I put the event on my calendar so that I would not forget.

Added example

後ろの人にも聞こえるように、大きな声で話してください。

Please speak loudly so that people in the back can hear.

Added example

子どもが触らないように、薬を高い所に置いた。

I put the medicine up high so that the child would not touch it.

Added example

日本語でニュースが読めるように、毎日漢字を勉強している。

I study kanji every day so that I can read the news in Japanese.

Book grammar · demand and request

～ように…・～ようにと…・～よう…

Book meaning

⇒要求することを表す

Expresses a demand or request.

Book formation

動　辞書形／ない形 ＋ように・ようにと・よう

＊③④のように、ていねいなお願いの場合はます形を使うこともある。

Book example

①を買ってくるように頼まれた。

Book example

②電車の中では携帯電話で話さないようにとされた。

Book example

③今週中にごくださいますよう、お願い申し上げます。

Book example

④試験にできますように。（祈るときの言い方）

Book cross-references

～ようだ　かぜをひいたようだ。のどが痛い。　→第１部G
～ようになる　→第１部G
～ようになっている　→第１部G
～ようにする・～ようにしている　→第１部11課－②、第１部G

Added examples

Added example

先生は学生に宿題を金曜日までに出すように言った。

The teacher told the students to submit the homework by Friday.

Added example

係員からここに荷物を置かないよう注意された。

An attendant warned me not to leave luggage here.

Added example

皆様が安全に帰れますよう、お祈り申し上げます。

We sincerely pray that everyone can return safely.

Book one-point lesson · page 73

「～ように」と「～ために」

◎（　）の中に「ように」か「ために」を書きなさい。

1　汚れをきれいに落とす（　）、特別なを使ってみた。

2　汚れがきれいに落ちる（　）、特別な洗剤を使ってみた。

3　この本は、簡単に漢字が覚えられる（　）、説明が工夫されている。

4　漢字をしっかり覚える（　）、一つの漢字を何度も紙に書いた。

5　覚えた漢字を忘れない（　）、ときどきしている。

6　かぜが早く治る（　）、ビタミンCをたくさんとっています。

7　自分の店を持つという夢をする（　）、会社を辞めた。

8　太陽の光が部屋いっぱいに入る（　）、カーテンを大きく開けてください。

9　大学に入る（　）、わたしはたくさんの準備をしなければならなかった。

10　子どもがいたずらをしない（　）、書類をきちんと片付けておこう。

Book explanation · ～ように

そうなってほしい状態。話者の意志を含まない動詞（無意志動詞・可能の意味の動詞・三人称が主語になる動詞など）につく。動詞の辞書形・ない形につく。

Expresses preference that a particular thing happens. Affixed to verbs that do not include the intention of the speaker (non-volitional verbs, verbs expressing possibility, and verbs for which the subject is a third person). Affixed to the dictionary form of the verb and its ない form.

Book example

・試合でいい成績が残せるように、みんながんばって練習している。

Book example

・子どもがたくさん野菜を食べるように、料理をいろいろ工夫している。

Book explanation · ～ために

そうしようという行為の目的。話者の意志を含む動詞の辞書形につく。

Indicates the purpose of an action. Affixed to the dictionary form of a verb expressing the will of the speaker.

Book example

・試合でいい成績を残すために、みんなががんばって練習している。

Book example

・食材をむだなく食べるために、料理をいろいろ工夫している。
