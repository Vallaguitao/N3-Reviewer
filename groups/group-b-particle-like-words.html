Book grammar · Topic

1. ～について…

Book formation

名　＋について

Book explanation

「～の内容・～に関係があることを…する。」～は話題を表す言葉。…は「思う・考える・話す・聞く・調べる・書く・説明する・知っている」などの動詞の文。

“… about, regarding …;” introduces a topic. The … phrase uses verbs such as おもう, かんがえる, はなす, きく, しらべる, かく, せつめいする or しっている.

Book example

①すみません、について聞きたいのですが……。

Book example

②今、わたしの国のについてレポートを書いています。

Book example

③この作品についてのを話していただけませんか。

Added examples

Added example

日本の祭りについて調べています。

I am researching Japanese festivals.

Added example

先生に試験について質問しました。

I asked the teacher about the exam.

Book grammar · Target and contrast

2. ～に対して…・～に対する

Book formation

名　＋に対して
名　＋に対する＋名

Book explanation

「～に向けて…する・…という態度を示す。」…は～への行為や態度などを表す文（求める・文句を言う・感謝する・きびしい・親切だなど）。

No real English equivalent. Means “do something vis-à-vis, regarding,” or “have an attitude regarding;” used with words such as もとめる, もんくをいう, かんしゃする, きびしい or しんせつだ.

Book example

A①ホテルの人は客に対して非常にていねいな言葉を使う。

Book example

A②父の意見に対して家族のみんながした。

Book example

A③新しくを作るため、会社側は近所のに対して理解を求めた。

Book example

A④最近、政府に対するが大きくなっている。

Book cross-reference

B　今年、３月は雨の日が多かったのに対して、４月は少なかった。

→第１部 ４課－①

Added examples

Added example

会社は利用者に対して説明する責任がある。

The company has a responsibility to explain things to its users.

Added example

弟は家族に対していつも親切だ。

My younger brother is always kind toward the family.

Book grammar · Agent and cross-references

3. ～によって…

Book formation

名　＋によって

Book explanation

…は受身文。～はその行為をする人。主に生物以外のものが主語になる受身文の中で、行為をする人を言いたいときに使う。

Denotes passive voice (was ... by ~); ~ indicates the agent. In passive sentences where the implied subject is inanimate, によって is often used to indicate the agent.

Book example

A①『坊っちゃん』という小説は1906年に夏目漱石によって書かれた。

Book example

A②このな祭りは昔からこの地方の人々によって守られてきた。

Book example

A③あるの人によって作られたこの歌を、今ではみんなが歌っている。

Book cross-reference

B　人によってが違う。

→第１部 ２課－②

Book cross-reference

C　タクシー代の値上げによって利用者が減った。

→第１部 ５課－②

Added examples

Added example

この建物は有名な建築家によって設計された。

This building was designed by a famous architect.

Added example

この曲は若い作曲家によって作られた。

This piece was composed by a young composer.

Book grammar · Standpoint

4. ～にとって…

Book formation

名　＋にとって

Book explanation

「～の立場から考えると…だ。」…は形容詞を含む文が多い。

as far as ~ is concerned, ... ... is often a phrase containing an adjective.

Book example

①日本に住むにとってはな問題だ。

Book example

②若い女性にとって買い物は楽しいことです。

Book example

③これはただの石ですが、わたしにとっては忘れられないの品です。

Added examples

Added example

わたしにとって日本語の会話が一番難しい。

For me, Japanese conversation is the most difficult.

Added example

家族にとって健康が何より大切だ。

For the family, health is more important than anything.

Book grammar · Role or capacity

5. ～として…

Book formation

名　＋として

Book explanation

「～という立場、資格、役割、名目で…」

Means “from the viewpoint of, in the capacity of, in the role of, on behalf of, as a/the …;”

Book example

①リーさんは留学生として日本に来た。

Book example

②大山君はこの学校のとして「を考える会議」に参加する。

Book example

③わたしはコーヒーカップを花びんとして使っています。

Book example

④この地方はお茶のとして有名です。

Added examples

Added example

この部屋は会議室として使われている。

This room is used as a meeting room.

Added example

田中さんは通訳として会議に参加した。

Mr. Tanaka took part in the meeting as an interpreter.

Book one-point lesson

「～について」と「～に対して」と「～にとって」

Book check

◎□から最も適当なものを選びなさい。

a　について　　b　にとって　　c　に対して

1　新しく来た先生（　）みんながうわさをしている。

2　先生（　）その話し方は失礼だ。

3　青山先生（　）学生たちは自分の子どものようなものだそうだ。

4　今のわたし（　）必要なのは、静かに考える時間だ。

5　あの（　）最も難しいのは、計画を住民に理解してもらうことだろう。

6　あの市長（　）何かご存じですか。

7　わたしたちはあの市長（　）計画のを求めた。

8　あの市長はわたしたち市民（　）いつも偉そうな態度だ。

Book explanation

～について：～は思考に関係のある行為（思う・書く・話す・聞くなど）の内容を表す。

The ~ phrase describes something you have thought, heard or otherwise processed mentally (おもう, かく, はなす, きく, etc).

Book example

例　親について文句を言う。（文句の内容は親に関係があること）

Book explanation

～に対して：～は行為（要求する・行うなど）や態度（親切だ・きびしいなど）が向けられる対象を表す。

The ~ phrase expresses the object of a demand or action (ようきゅうする, おこなう, etc), or of an attitude or judgment (しんせつだ or きびしい, etc).

Book example

例　親に対して文句を言う。（文句を言う相手は親）

Book explanation

～にとって：～は判断や評価をする立場を表す。

The ~ phrase indicates the person making a judgment or evaluation.

Book example

例　親にとって子どもの成長は何よりの喜びだ。（喜びだと考える人は親）
