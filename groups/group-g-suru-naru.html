Book overview · intention and result

「する」と「なる」の使い分けの基本

するなる
Book explanation

する：に注目

Focuses on the intentional behavior of the speaker or other person.

Book example

わたしは部屋をきれいにした。

Book explanation

なる：に注目

Focuses on change in state or result.

Book example

部屋はきれいになった。

Expanded explanation

Use する when an agent deliberately causes or chooses something. Use なる when the sentence presents the resulting state or change, even if a person caused it in the background.

Added examples

Added example

母は部屋を明るくした。

My mother made the room bright.

Added example

部屋が明るくなった。

The room became bright.

Book comparison · changing a state

～にする・～くする ／ ～になる・～くなる

するなる
Book form

～にする・～くする
⇒状態を変える

Book explanation

Means to change a state or situation.

Book example

わたしは大きいケーキを半分にした。

Book example

を食べて、骨を丈夫にしたい。

Book example

電気を消して、部屋を暗くしてください。

Book form

～になる・～くなる
⇒状態が変わる

Book explanation

Used when a state or situation changes of itself.

Book example

大きいケーキが半分になった。

Book example

小魚をよく食べたので、骨が丈夫になった。

Book example

電気を消したので、部屋が暗くなった。

Book example

最近、この川では魚があまり釣れなくなった。

Added examples

Added example

音を小さくしてください。

Please turn the sound down.

Added example

音が小さくなった。

The sound became quieter.

Added example

会議室を静かにした。

I made the meeting room quiet.

Added example

会議室が静かになった。

The meeting room became quiet.

するなる
Book form

⇒決める

Book explanation

Decide (with person as agent).

Book comparison · continuing patterns

～にしている・～ことにしている ／ ～になっている・～ことになっている

するなる
Book form

～にしている・～ことにしている
⇒決めたことを続けている

Book explanation

Continue with a pre-established pattern of action.

Book example

昼食はいつもパンにしています。

Book example

寝る前に必ず日記を書くことにしている。

Book example

はもらわないことにしている。

Book form

～になっている・～ことになっている
⇒決まったことが続いている（決まり・予定）

Book explanation

Pre-established pattern of behavior continues(decided or planned).

Book example

Book example

この会社ではを着ることになっている。

Added example

I keep fruit as my breakfast choice.

Added example

夜はコーヒーを飲まないことにしている。

I make it a rule not to drink coffee at night.

Added example

受付は一階になっています。

The reception desk is set up on the first floor.

Added example

図書館では電話を使わないことになっている。

The rule is that phones are not used in the library.

Book comparison · purpose, arising change, and design

～ようにする A ／ ～ようになる・～ようになっている

するなる
Book form

～ようにする　A
⇒ある目的のために変化を起こす

Book explanation

Cause a change for a specific purpose.

Book example

机の位置を変えて、仕事中でも外の景色が見えるようにしよう。

Book example

ドアに穴を空けて、ねこが通れるようにした。

Book example

大事な物はいつも棚の上に置いて、子どもに触られないようにしている。

Book form 1

～ようになる
⇒変化が起きる

Book explanation

A change arises of itself.

Book example

机の位置を変えたので、仕事中でも外の景色が見えるようになった。

Book example

親が楽しそうに家事をしていれば、子どもも進んで手伝うようになる。

Book form 2

～ようになっている
⇒ある目的のためにそう作られている

Book explanation

Means done or arranged in such a way for a specific purpose.

Book example

この家は屋根にも窓があって、の光が上からも入るようになっている。

Book example

トイレに入ると、電気がでつくようになっている。

Book example

この学校の音楽室は、楽器の音が外の人には聞こえないようになっています。

Added examples

Added example

字幕を付けて、言葉がわかるようにした。

I added subtitles so the words could be understood.

Added example

練習して、漢字が読めるようになった。

Through practice, I became able to read kanji.

Added example

このドアは押すと開くようになっている。

This door is designed to open when pushed.

Added example

子どもが届かないように、薬を高い所に置いた。

I put the medicine up high so children could not reach it.

Book grammar · habitual care

～ようにする B・～ようにしている

Book form

～ようにする　B
～ようにしている
⇒習慣的に心がける

Book explanation

Habitually take care to/not to.

Book example

暑い日には十分を取るようにしましょう。

Book example

雪の日は車を運転しないようにしている。

Book example

メールにはすぐするようにしている。

Expanded explanation

This B use describes a conscious repeated effort. It does not state that the behavior happens automatically; the speaker keeps trying to make it a habit.

Added examples

Added example

毎日新聞を読むようにしている。

I make a point of reading the newspaper every day.

Added example

約束に遅れないようにしています。

I take care not to be late for appointments.

Added example

疲れたら早く寝るようにしましょう。

Let us make a point of going to bed early when tired.

Added example

新しい言葉を会話で使うようにしている。

I make an effort to use new words in conversation.

Book exercises

練習１　どちらか適当な方を選びなさい。

1　スープの味が濃かったから、お湯を入れて薄く（a した　b なった）。

2　りんご（a を赤くしたら　b が赤くなったら）、木から取って食べてもいいよ。

3　テレビで紹介されたので、この町の祭りは（a 有名にした　b 有名になった）。

4　【美容院で】今日は10センチぐらい（a 短くして　b 短くなって）ください。

5　A「お子さんのかぜ、いかがですか。」 B「はい、だいぶよく（a しました　b なりました）。」

6　図書館では古い本は捨てられることに（a した　b なった）。

7　わたしはあした退院できることに（a しました　b なりました）。

8　体重を減らしたいから、ご飯は毎食茶わん１杯だけに（a しよう　b なろう）。

9　今度の冬のオリンピックはどこに（a しましたか　b なりましたか）。

10　わたしは大事なことは何でもメモすることに（a している　b なっている）。

11　この公園では花火をしてはいけないことに（a している　b なっている）そうだ。

12　子どもは何歳で歩けるように（a するんですか　b なるんですか）。

13　食べ物はよくかんで食べるように（a しましょう　b なりましょう）。

14　ねこの目は、入ってくる光の量を調節できるように（a している　b なっている）。

15　先生「大切な書類だから、書き間違えないように（a すること　b なること）。」

練習２　どちらか適当な方を選びなさい。

Book dialogue

A「リンさん、こんにちは。あれ、どうして部屋を片付けているんですか。」

B「引っ越しする（① a ことに　b ように）したんです。来月から弟といっしょに住む（② a ことに　b ように）なったので……。二人で住むのには、ここはちょっと狭いんですよ。今度の所は会社からだいぶ遠くなりますけど。」

A「じゃ、寝坊しない（③ a ことに　b ように）しなければね。」

B「ええ。駅からも遠いので、自転車を買う（④ a ことに　b ように）しました。」

A「引っ越しはいつなんですか。」

B「急なんですが、あしたの朝９時に引っ越しのトラックが来る（⑤ a ことに　b ように）なったんです。」

A「それは大変ですね。手伝いましょうか。」

B「いえ、大丈夫です。今から弟と弟の友だちが手伝いに来る（⑥ a ことに　b ように）なっていますから。いろいろお世話になりました。」

①　引っ越しする（　）したんです。

②　来月から弟といっしょに住む（　）なったので……。

③　じゃ、寝坊しない（　）しなければね。

④　自転車を買う（　）しました。

⑤　引っ越しのトラックが来る（　）なったんです。

⑥　弟と弟の友だちが手伝いに来る（　）なっていますから。

Book one-point lesson · page 87

「～ようにしている」と「～ようになっている」

◎どちらか適当な方を選びなさい。

1　わたしはなるべく肉よりも（a 魚を食べる　b 魚が食べられる）ようにしている。

2　うちの台所は、長い時間ガスを使うと（a 火を消す　b 火が消える）ようになっている。

3　ホテルでは、お風呂に入っていても（a 電話をかける　b 電話がかけられる）ようになっている。

4　このおもちゃは面白い。手をたたくと（a 人形を動かす　b 人形が動く）ようになっている。

5　健康のためになるべく（a 体を動かす　b 体が動く）ようにしてください。

6　このドアは内側からは外が見えるが、外からは（a 中を見ない　b 中が見えない）ようになっている。

7　新聞のテレビ番組表を見るとテレビが見たくなるので、試験前は（a 見ない　b 見えない）ようにしているんです。

Book explanation · ～ようにしている

～ようにしている：主語はふつう「話者」。～には意志動詞を使う。

The speaker is normally the subject. ～ is a volitional verb.

Book example

例　わたしはいつも折りたたみのかさを持ち歩くようにしています。

Book explanation · ～ようになっている

～ようになっている：主語はふつう、人間ではない。～には意志を含まない動詞（可能を表す動詞や三人称が主語の動詞など）を使う。

Subject is not usually a person. ～ uses a non-volitional verb (verbs expressing potential and verbs with a third person subject, etc.).

Book example

例　このいすは軽くて小さく折りたためるので、楽に持ち運べるようになっています。
