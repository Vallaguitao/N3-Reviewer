Book overview

ある人が言ったこと、自分が思ったことの内容を、文の中に入れるには、次のような方法があります。

The following methods are used to include a statement by somebody else, or one's own impressions, in a sentence.

Book source · pattern 1

普通形＋と…「言う・思う・感じる」など

Book formation

普通形＋と…「言う・思う・感じる」など

Expanded explanation

How the sentence is built

Put the quoted thought or statement into plain form before と. The final reporting verb carries the tense and politeness for the whole sentence.

Common mistake

Do not keep でしょう inside the embedded clause when the surrounding report requires the plain equivalent だろう.

Book examples

Book example

わたしは思います。＋「リンさんはきっとするでしょう。」

→

わたしはリンさんはきっとするだろうと思います。

Added examples

Added example

田中さんは明日来ると言いました。

Mr. Tanaka said that he would come tomorrow.

Why this works

The quoted content 明日来る is plain before と言いました.

Added example

この方法がいちばん安全だと思います。

I think this method is the safest.

Why this works

A noun or な-adjective statement uses だ inside a plain quotation.

Book source · pattern 2

「辞書形・ない形」＋ように（＋と）／「て形・ない形＋で」＋ほしい＋と／命令形・禁止形＋と

Book formation

「辞書形・ない形」＋ように（＋と）／［て形・ない形＋で］＋ほしい＋と／命令形・禁止形＋と　…「頼む・注意する・言う」など

Expanded explanation

How the sentence is built

Use ように for an indirect instruction, てほしい for the action you want another person to perform, and the imperative or prohibitive when reporting a direct command.

Common mistake

Keep the direction of the request clear: the person marked by に is the person asked, advised, or ordered to act.

Book examples

Book example

医者はカンさんに。＋「ときどきをしたほうがいいです」

→

医者はカンさんにときどきをするように。

Book example

わたしは先生に。＋「もう一度してくださいませんか」

→

わたしは先生にもう一度してほしいと。

Book example

母はわたしに言った。＋「お金をに使いなさいね」

→

母はわたしにお金をに使えと言った。

Added examples

Added example

先生は学生に辞書を持ってくるように言いました。

The teacher told the students to bring a dictionary.

Why this works

ように言う reports an instruction without using a direct imperative.

Added example

店長に来週休ませてほしいと頼みました。

I asked the manager to let me take next week off.

Why this works

てほしいと頼む states the action requested from the manager.

Book source · pattern 3

普通形＋かどうか／疑問詞＋～か

Book formation

普通形＋かどうか／疑問詞＋～か　…「聞く・わからない・調べる」など

Expanded explanation

How the sentence is built

Use かどうか for a yes-or-no issue. When the original question contains だれ・何・いつ・どこ・どう, keep that question word and end the embedded clause with か.

Common mistake

Do not add かどうか after a question word; use いつ来るか, not いつ来るかどうか.

Book examples

Book example

わたしはカンさんに。＋「あした会えるでしょうか」

→

わたしはカンさんにあした会えるかどうか（を）。

Book example

わたしはわからない。＋「どうすれば日本語がになるでしょうか」

→

わたしはどうすれば日本語がになるか（が）わからない。

Added examples

Added example

会議が何時に始まるか調べました。

I checked what time the meeting begins.

Why this works

何時 is retained and the embedded question ends in か.

Added example

この答えが正しいかどうかわかりません。

I do not know whether this answer is correct.

Why this works

かどうか frames the yes-or-no uncertainty.

Book source · pattern 4

どうして／なぜ～かというと、…からだ

Book formation

どうして／なぜ～かというと、…からだ。

→第３部　１課

Expanded explanation

How the sentence is built

This fixed frame first turns the why-question into an embedded clause ending in かというと, then answers it with a reason ending in からだ.

Common mistake

The clause before かというと uses plain form: 人気がある, not 人気があります.

Book examples

Book example

「どうしてこの店はがありますか」＋安くておいしいからです。

→

どうしてこの店ががあるかというと、安くておいしいからです。

Added examples

Added example

なぜ日本語を勉強しているかというと、日本で働きたいからです。

Why am I studying Japanese? Because I want to work in Japan.

Why this works

The first half raises the embedded why-question; the second gives the reason.

Added example

どうして遅れたかというと、電車が止まったからです。

Why was I late? Because the train stopped.

Why this works

Both clauses are plain inside the fixed explanatory frame.
