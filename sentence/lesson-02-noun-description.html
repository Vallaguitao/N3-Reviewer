Book overview

名詞を説明する形式はいろいろありますが、説明する言葉は必ず名詞の前に来ます。

There are several ways of modifying nouns, but the modifier must come before the noun.

Book source · pattern 1

［普通形（ナ形だ→な・名だ→の）］＋名

Book formation

［普通形（ナ形だ→な・名だ→の）］＋名

Expanded explanation

How the sentence is built

Move the entire plain-form clause before the noun. The described noun replaces the original topic or missing element, so no extra の is inserted after a verb clause.

Common mistake

Do not put the modifier after the noun. Japanese places even long explanatory clauses immediately before the noun.

Book examples

Book example

そのバスはに行く

→

に行くバス

Book example

そのバスに乗ると子どもが

→

子どもがバス

Book example

そのニュースを聞いてショックを

→

ショックをニュース

Book example

にぎやかなで子どもたちが話している

→

子どもたちが話しているにぎやかな

Added examples

Added example

昨日駅で会った人は田中さんです。

The person I met at the station yesterday is Mr. Tanaka.

Why this works

昨日駅で会った directly modifies 人.

Added example

料理が上手な友だちに教えてもらいました。

A friend who is good at cooking taught me.

Why this works

A な-adjective uses な before the modified noun.

Book source · pattern 2

［助詞］＋の＋名

Book formation

［助詞］＋の＋名

Expanded explanation

How the sentence is built

Particles such as まで, から, and へ can combine with の to turn a relationship into a noun modifier.

Common mistake

Choose the particle for the original relationship first, then add の; do not replace every particle with の.

Book examples

Book example

そのバスはまで行く

→

までのバス

Book example

母からが来た

→

母からの

Book example

友だちにプレゼントをあげる

→

友だちへのプレゼント

Added examples

Added example

先生からのメールを読みました。

I read the email from the teacher.

Why this works

からの preserves the source relationship before メール.

Added example

京都までの切符を買いました。

I bought a ticket as far as Kyoto.

Why this works

までの marks the ticket's destination limit.

Book source · pattern 3

［助詞のような働きをする言葉］＋の＋名

Book formation

［助詞のような働きをする言葉］＋の＋名

→第１部Ｂ

Expanded explanation

How the sentence is built

Multiword expressions that act like particles can take の before a noun, preserving their original relationship.

Common mistake

Use the complete particle-like expression before の; omitting a required part changes the relationship.

Book examples

Book example

そのバスをとして使う

→

としてのバス

Book example

について本を書いた

→

についての本

Added examples

Added example

環境についての意見を聞きました。

I heard an opinion about the environment.

Why this works

についての keeps the topic relation before 意見.

Added example

学生としての責任を忘れないでください。

Please do not forget your responsibilities as a student.

Why this works

としての marks the role attached to 責任.

Book source · pattern 4

［助詞のような働きをする言葉の名詞につく形］＋名

Book formation

［助詞のような働きをする言葉の名詞につく形］＋名

→第１部Ｂ

Expanded explanation

How the sentence is built

Some particle-like expressions have a dedicated form used directly before nouns, such as による and に対する.

Common mistake

Before a noun use による or に対する, not sentence-ending によって or に対して.

Book examples

Book example

そのバスは市のによって走っている

→

市のによるバス

Book example

Ａ案に対してを言う

→

Ａ案に対する

Added examples

Added example

台風による被害が広がっています。

Damage caused by the typhoon is spreading.

Why this works

による directly modifies 被害.

Added example

新しい規則に対する質問があります。

I have a question about the new rule.

Why this works

に対する is the noun-modifying form before 質問.

Book source · pattern 5

［状態や様子を表す言葉（ばかり・とおり・まま・はずなど）］＋の＋名

Book formation

［状態や様子を表す言葉（ばかり・とおり・まま・はずなど）］＋の＋名

Expanded explanation

How the sentence is built

A phrase that describes a state or expected condition can take の before the noun whose state is being identified.

Common mistake

Keep the full state expression—ばかり, とおり, まま, or はず—before の; do not attach の directly to the verb when the state word is required.

Book examples

Book example

そのバスはしたばかりだ

→

したばかりのバス

Book example

そのはガイドブックで見たとおりだった

→

ガイドブックで見たとおりの

Book example

この服はままだ

→

ままの服

Added examples

Added example

届いたばかりの荷物を開けました。

I opened the package that had just arrived.

Why this works

届いたばかりの describes the package's recent state.

Added example

昔のままの町並みが残っています。

The townscape remains just as it was long ago.

Why this works

昔のままの identifies an unchanged state.
