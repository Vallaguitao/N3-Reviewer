Book source · pattern 1

［普通形（ナ形だ→な・名だ→の）］＋名

Book formation

［（ナだ→な・だ→の）］＋

[Normal form (Na form → na/name da → no)] + first name

Expanded explanation

How the sentence is built

Move the entire plain-form clause before the noun. The described noun replaces the original topic or missing element, so no extra の is inserted after a verb clause.

Common mistake

Do not put the modifier after the noun. Japanese places even long explanatory clauses immediately before the noun.

Book examples

Book example

そのバスはに

The bus goes to Tokyo Station

→

にバス

Bus going to Tokyo Station

Book example

そのバスにとが

Children will be happy when they ride that bus.

→

がバス

A bus that kids will love

Book example

そのニュースをショックを

I was shocked when I heard the news.

→

ショックをニュース

shocking news

Book example

にぎやかなでたちがいる

Children are talking in loud voices

→

たちがいるにぎやかな

The lively voices of children talking

Added examples

Added example

ではさんです。

The person I met at the station yesterday is Mr. Tanaka.

Why this works

昨日駅で会った directly modifies 人.

Added example

がなにもらいました。

A friend who is good at cooking taught me.

Why this works

A な-adjective uses な before the modified noun.

Book source · pattern 2

［助詞］＋の＋名

Book formation

［］＋の＋

[Particle] + noun of +

Expanded explanation

How the sentence is built

Particles such as まで, から, and へ can combine with の to turn a relationship into a noun modifier.

Common mistake

Choose the particle for the original relationship first, then add の; do not replace every particle with の.

Book examples

Book example

そのバスはまで

The bus goes to the station

→

までのバス

bus to the station

Book example

からが

I received a letter from my mother

→

からの

letter from mother

Book example

にプレゼントをあげる

give a gift to a friend

→

へのプレゼント

gifts for friends

Added examples

Added example

からのメールを。

I read the email from the teacher.

Why this works

からの preserves the source relationship before メール.

Added example

までのを。

I bought a ticket as far as Kyoto.

Why this works

までの marks the ticket's destination limit.

Book source · pattern 3

［助詞のような働きをする言葉］＋の＋名

Book formation

［のようなをする］＋の＋

[Words that function like particles] + name of +

→１Ｂ

→Part 1B

Expanded explanation

How the sentence is built

Multiword expressions that act like particles can take の before a noun, preserving their original relationship.

Common mistake

Use the complete particle-like expression before の; omitting a required part changes the relationship.

Book examples

Book example

そのバスをとして

use the bus as a means of commuting to school

→

としてのバス

Bus as a means of commuting to school

Book example

についてを

wrote a book about life

→

についての

books about life

Added examples

Added example

についてのを。

I heard an opinion about the environment.

Why this works

についての keeps the topic relation before 意見.

Added example

としてのをください。

Please do not forget your responsibilities as a student.

Why this works

としての marks the role attached to 責任.

Book source · pattern 4

［助詞のような働きをする言葉の名詞につく形］＋名

Book formation

［のようなをするのにつく］＋

[Form attached to a noun of a word that acts like a particle] + first name

→１Ｂ

→Part 1B

Expanded explanation

How the sentence is built

Some particle-like expressions have a dedicated form used directly before nouns, such as による and に対する.

Common mistake

Before a noun use による or に対する, not sentence-ending によって or に対して.

Book examples

Book example

そのバスはのによっている

The bus is operated by the city.

→

のによるバス

Buses operated by the city

Book example

Ａを

express an opposing opinion to Plan A

→

Ａ

Opposition to Plan A

Added examples

Added example

によるがいます。

Damage caused by the typhoon is spreading.

Why this works

による directly modifies 被害.

Added example

があります。

I have a question about the new rule.

Why this works

に対する is the noun-modifying form before 質問.

Book source · pattern 5

［状態や様子を表す言葉（ばかり・とおり・まま・はずなど）］＋の＋名

Book formation

［やを（ばかり・とおり・まま・はずなど）］＋の＋

[Words that express a state or situation (bakashi, orori, mama, hazu, etc.)] + name of +

Expanded explanation

How the sentence is built

A phrase that describes a state or expected condition can take の before the noun whose state is being identified.

Common mistake

Keep the full state expression—ばかり, とおり, まま, or はず—before の; do not attach の directly to the verb when the state word is required.

Book examples

Book example

そのバスはしたばかりだ

The bus has just been repaired

→

したばかりのバス

Newly repaired bus

Book example

そのはガイドブックでとおりだった

The scenery was just like I saw in the guidebook.

→

ガイドブックでとおりの

Scenery exactly as seen in the guidebook

Book example

このはままだ

These clothes are still dirty

→

ままの

dirty clothes

Added examples

Added example

ばかりのを。

I opened the package that had just arrived.

Why this works

届いたばかりの describes the package's recent state.

Added example

のままのがいます。

The townscape remains just as it was long ago.

Why this works

昔のままの identifies an unchanged state.
